================================================================================
                      KEY FEATURES
================================================================================

✅ FAIR COMPARISON:
   Both models use identical preprocessing:
   - Load at 8kHz
   - Random volume scaling (0.7x - 1.0x)
   - Gaussian noise (σ=0.005)
   - Clip to [-1.0, 1.0]
   - Normalize
   - Pad/truncate to 4 seconds

✅ CLASS IMBALANCE HANDLING:
   WeightedRandomSampler ensures minority class (deepfakes)
   is properly represented during training

✅ EARLY STOPPING:
   Wav2Vec2 training stops if no improvement for 3 epochs
   (prevents overfitting, saves time)

✅ EER IMPLEMENTATION:
   Proper computation of Equal Error Rate:
   - Find threshold where FAR = FRR
   - Lower EER indicates better overall performance
   - More robust than accuracy for security applications

✅ GPU/CPU SUPPORT:
   Automatic device detection with fallback to CPU
   (GPU recommended but not required)

✅ MODEL CHECKPOINTING:
   Best model saved based on validation F1 score

✅ COMPREHENSIVE LOGGING:
   Progress reporting, epoch-by-epoch metrics,
   training/validation losses and accuracies
