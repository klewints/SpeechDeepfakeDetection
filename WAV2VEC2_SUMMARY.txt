================================================================================
KEY FEATURES IMPLEMENTED
================================================================================

1. SHARED TELEPHONY PREPROCESSING
   Both models use identical preprocessing:
   - Load audio at 8kHz sample rate
   - Random volume scaling (0.7x - 1.0x)
   - Add Gaussian noise (μ=0, σ=0.005)
   - Clip to [-1.0, 1.0]
   - Normalize by max amplitude
   
   This ensures fair comparison - performance difference is due to
   feature extraction, not preprocessing.

2. FIXED DURATION PADDING
   All audio normalized to 4 seconds (32,000 samples):
   - CNN: Mel Spectrogram (1, 128, T)
   - Wav2Vec2: Raw waveform (32,000,)
   Ensures consistent batch processing.

3. CLASS IMBALANCE HANDLING
   WeightedRandomSampler in both training scripts:
   - Automatic weight calculation: 1.0 / class_count
   - Balances rare class (deepfakes) during training
   - Improves model performance on imbalanced datasets

4. COMPREHENSIVE EVALUATION METRICS
   
   ✓ Accuracy: (TP+TN)/(Total) — Overall correctness
   ✓ Precision: TP/(TP+FP) — False positive rate
   ✓ Recall: TP/(TP+FN) — False negative rate
   ✓ F1 Score: Harmonic mean of Precision & Recall
   ✓ ROC-AUC: Area under ROC curve (0.5→1.0)
   ✓ EER: Equal Error Rate (FAR=FRR) — MOST IMPORTANT
   ✓ Confusion Matrix: TP, TN, FP, FN counts

   EER is the threshold where False Acceptance Rate = False Rejection Rate.
   Lower EER indicates better overall system performance.

5. VISUALIZATION SUITE
   ✓ Confusion matrix heatmaps (both models)
   ✓ ROC curve comparison plot
   ✓ Results table (CSV export)

================================================================================
ARCHITECTURE COMPARISON
================================================================================

CNN + MEL SPECTROGRAM:
  Input: Raw audio → Mel Spectrogram (128×T)
  Feature Learning: Fixed (librosa STFT)
  Architecture: 2 Conv blocks + classifier
  Parameters: ~100K
  Inference: Fast (~10ms)
  
WAV2VEC2:
  Input: Raw waveform (32K,)
  Feature Learning: Transformer-based (pretraining)
  Architecture: Frozen encoder + classifier head
  Parameters: ~360M (mostly frozen)
  Inference: Slower (~100ms)
  
ADVANTAGE: Wav2Vec2 learns from 53k hours of unlabeled speech data,
capturing phonetic and speaker information robust to deepfake artifacts.
