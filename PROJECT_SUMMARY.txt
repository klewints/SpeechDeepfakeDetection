## 🎯 Features Implemented

### ✅ Dual Pipeline Architecture
```
┌─────────────────────────────────────────────────────────────────┐
│                    AUDIO INPUT (any duration)                   │
│                            ↓                                     │
│              Load at 8kHz + Telephony Effects                    │
│              (volume scaling + noise + clipping)                 │
│                            ↓                                     │
│        ┌──────────────────────────────────────────┐              │
│        │         4-second fixed padding           │              │
│        └──────────────────────────────────────────┘              │
│        ↙                                      ↘                  │
│    (32,000)                               (1, 128, T)           │
│    Waveform                            Mel Spectrogram          │
│        ↓                                      ↓                  │
│   Wav2Vec2                               CNN                    │
│  Classifier                          Classifier                 │
│        ↓                                      ↓                  │
│   Output: (2,)                         Output: (2,)             │
│  [real, fake]                         [real, fake]             │
└─────────────────────────────────────────────────────────────────┘
```

### ✅ Shared Preprocessing (Identical for Both Models)
- Load audio at 8kHz
- Random volume scaling (0.7x - 1.0x)
- Gaussian noise (μ=0, σ=0.005)
- Clip to [-1.0, 1.0]
- Normalize by max amplitude
- Pad/truncate to 4 seconds

### ✅ Seven Evaluation Metrics
1. **Accuracy** — Overall correctness
2. **Precision** — False positive rate
3. **Recall** — False negative rate
4. **F1 Score** — Harmonic mean
5. **ROC-AUC** — Area under ROC curve
6. **EER** — Equal Error Rate (MOST IMPORTANT)
7. **Confusion Matrix** — TP, TN, FP, FN

### ✅ Comprehensive Visualizations
- Confusion matrix heatmaps (both models)
- ROC curve comparison plot
- Results exported to CSV

### ✅ Model Features
- **CNN**: 2 conv blocks + classifier (100K params)
- **Wav2Vec2**: Frozen pretrained encoder + classifier (360M params, mostly frozen)
- Class imbalance handling (WeightedRandomSampler)
- Early stopping on validation F1
- GPU support with automatic fallback to CPU

## 🎓 Key Implementation Highlights

### 1. Fair Comparison
Both models use **identical preprocessing**. Performance differences reflect feature extraction quality, not preprocessing differences.

### 2. EER Computation
```python
def compute_eer(y_true, y_scores):
    fpr, tpr, thresholds = roc_curve(y_true, y_scores)
    frr = 1 - tpr
    far = fpr
    diff = np.abs(far - frr)
    eer_idx = np.argmin(diff)
    eer = (far[eer_idx] + frr[eer_idx]) / 2
    return eer, threshold[eer_idx]
```

EER is the threshold where False Acceptance Rate = False Rejection Rate.

### 3. Class Imbalance Handling
```python
class_counts = np.bincount(train_labels)
class_weights = 1.0 / class_counts
sampler = WeightedRandomSampler(class_weights, len(class_weights))
dataloader = DataLoader(dataset, sampler=sampler)
```

Ensures minority class (deepfakes) is properly represented during training.

### 4. Transfer Learning
Wav2Vec2 encoder is **frozen** during training:
- Reduces memory (360MB → 100MB)
- Reduces training time
- Prevents overfitting on small datasets
- Still leverages 53k hours of pretraining
- Can be unfrozen for better results (requires more resources)

## 🔧 Technical Specifications

### Models
- **CNN**: Custom 2-layer CNN (100K params)
- **Wav2Vec2**: facebook/wav2vec2-base (360M params, frozen)

### Training Configuration
- **Optimizer**: Adam
- **Loss**: CrossEntropyLoss
- **Batch size**: CNN=16, Wav2Vec2=8
- **Epochs**: CNN=10, Wav2Vec2=15
- **Learning rate**: CNN=1e-5, Wav2Vec2=1e-4
- **Class balancing**: WeightedRandomSampler
- **Early stopping**: Yes (Wav2Vec2)
- **GPU support**: Automatic with fallback to CPU

### Data
- **Audio**: 8kHz, 4 seconds, [-1, 1] range
- **Labels**: 0=real, 1=fake
- **Split**: 80% train, 20% validation
- **Augmentation**: Telephony effects (shared)
